## Install

1. Clone repo:
```js
git clone https://github.com/webbobtba/counterdemoparcel.git
```

2. Go into directory:
```js
cd counterdemoparcel
```

3. Get dependencies:
```js
npm install
```

4. Start parcel:
```js
npm start
```
